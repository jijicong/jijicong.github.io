# Markdown -- 整理

###  一 、标题

``` markdown
1. 最高阶标题
This is an H1
=============

2. 第二阶标题
This is an H2
-------------

3. 用#表示阶数（只有1-6阶）
# 这是 H1
## 这是 H2
###### 这是 H6
```
1.
This is an H1
===============

2.
This is an H2
-------------

3.
# 这是 H1
## 这是 H2
###### 这是 H6

### 七 、 代码

